# 八字合婚（双盘对比）规则

## 概述

八字合婚是中国传统命理中判断两人婚姻匹配度的核心方法。合婚不是简单的「吉凶」判断，而是分析双方的互补性、潜在冲突点和调和方式。

---

## 一、合婚七大维度

### 1. 纳音婚配（年柱纳音对比）

年柱纳音相生为吉，相克需留意。

| 关系 | 含义 | 例 |
|------|------|-----|
| 相生 | 气场相生，和谐 | 泉中水生桑柘木（水生木） |
| 相克 | 气场冲突 | 泉中水克屋上土（水克土） |
| 相同 | 志趣相投但不互补 | 两柱皆为水 |

### 2. 日柱对比

日柱代表本人和配偶宫。双方的日柱关系是合婚核心：

- **天干五合**：如甲己合、乙庚合 → 性格相吸，强烈吸引
- **地支六合**：如子丑合、寅亥合 → 生活习惯契合
- **地支相冲**：如子午冲、卯酉冲 → 需要高度包容
- **地支相刑**：如子卯刑、寅巳申刑 → 潜在摩擦

### 3. 五行互补

分析双方命局中五行的强弱，判断互补程度：

- **对方旺你的弱**：最佳互补 → 如你缺火对方火旺
- **对方弱你的旺**：你滋养对方 → 你付出更多
- **对方旺你的旺**：同旺冲突 → 互相竞争
- **对方弱你的弱**：双双缺位 → 需外界补充

### 4. 十神配合

**男命看财星，女命看官星。日主代表自己。**

- 男命正财/偏财星有力 + 女命正官/七杀有力 → 传统佳配
- 男命财星被合/克 + 女命官星被合/克 → 婚姻需磨合
- 一方食伤重 + 另一方印星重 → 创意 × 务实的互补组合

### 5. 生肖婚配（民间最常用）

| 生肖 | 三合/六合（佳配） | 相冲（需注意） | 相害（需注意） |
|------|-----------------|--------------|--------------|
| 鼠(子) | 牛、龙、猴 | 马(午) | 羊(未) |
| 牛(丑) | 鼠、蛇、鸡 | 羊(未) | 马(午) |

### 6. 大运同步度

分析双方当前所处大运的五行特性是否协调：

- 双方同走财/官运 → 共同目标，齐头并进
- 一方走财一方走印 → 一人赚钱一人持家，互补
- 一方走劫财一方走财 → 财务观念冲突

### 7. 用神互补性

命局最需的五行，对方能否提供：

- 你的喜用神是对方的旺五行 → 最佳配置
- 你的忌神恰是对方的弱五行 → 至少不互伤
- 你的忌神是对方的旺五行 → 互相刺激对方弱点

## 二、综合评分框架

| 维度 | 权重 | 评分标准 |
|------|:---:|---------|
| 日柱关系 | 25% | 合>生>克>冲>刑 |
| 五行互补 | 25% | 互补度 |
| 纳音婚配 | 10% | 相生>相同>相克 |
| 十神配合 | 15% | 财官有力 |
| 生肖婚配 | 10% | 合>无害>冲>害 |
| 大运同步 | 10% | 同步度 |
| 用神互补 | 5% | 互补度 |

**总分建议**：
- 85+：天生佳偶
- 70-84：良好匹配，需要适当磨合
- 55-69：可考虑，需双方加倍努力
- 40-54：较大挑战，需深度沟通
- <40：建议谨慎考虑

## 四、注意事项

1. 合婚结果仅供文化参考，不可替代实际相处中的了解和沟通
2. 生肖婚配是民间简化版，应以后六项分析为主
3. 同一对夫妻在不同大运阶段的匹配感受可能不同
4. 真正的婚姻质量取决于双方的经营，而非八字「匹配」程度
