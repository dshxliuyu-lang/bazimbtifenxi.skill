# 五行补救方案详解

## 概述

当命局中某五行过弱或缺位，或某五行过旺需要制化时，可通过后天的方式进行调节和补救。补救不是"改变命运"，而是通过环境的微调让生活更顺畅。

---

## 一、五行补金

### 适合人群
- 命局缺金、金弱受克
- 喜用神为金

### 补救方案

| 类别 | 内容 |
|------|------|
| **颜色** | 白色、金色、银色 |
| **方位** | 西、西北 |
| **数字** | 4、9 |
| **行业** | 金融、银行、会计、五金、机械、法律、军警、精密制造、珠宝 |
| **饰品** | 金银首饰、金属手表、白水晶 |
| **饮食** | 白色食物：白萝卜、百合、银耳、梨、牛奶、白肉 |
| **衣着** | 多穿白色、浅灰、金属色系 |
| **音乐** | 金属乐器：钢琴、吉他、编钟 |

### 命理依据
《滴天髓》：「金主义，其性刚，其情烈。」金代表义气、规则、刚毅。补金可增强决断力和原则性。

## 六、补运时机指南

### 按大运选择
- 走**印星大运**时：抓住学习充电机会（金、水行业佳）
- 走**食伤大运**时：尽情创作输出（木、火行业佳）
- 走**财星大运**时：积极开源投资（火、土行业佳）
- 走**官杀大运**时：顺势承担更多责任，积累地位

### 按流年选择
流年是喜用神年份 → 大胆行动、做重大决策
流年是忌神年份 → 稳守、学习、积累、减少冒进

## 七、补救原则

1. **以喜用神为准**：不是缺什么补什么，而是喜什么补什么
2. **循序渐进**：微调环境即可，不需要大规模改变
3. **结合现实**：行业选择要结合个人能力和市场，不能只看五行
4. **内在修为同样重要**：修心养性、持续学习才是根本

> 「命理提供方向，努力决定高度。」
