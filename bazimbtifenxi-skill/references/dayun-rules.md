# 大运规则详解

## 一、阳年与阴年

天干分阴阳：
- **阳年**：年干为甲、丙、戊、庚、壬
- **阴年**：年干为乙、丁、己、辛、癸

## 二、大运顺逆规则

| 年干 | 性别 | 大运方向 |
|------|------|---------|
| 阳年 | 男 | 顺排 |
| 阳年 | 女 | 逆排 |
| 阴年 | 男 | 逆排 |
| 阴年 | 女 | 顺排 |

口诀：阳男阴女顺排，阴男阳女逆排。

## 三、大运排列方法

以月柱干支为基准，进行顺排或逆排：

### 顺排
从月柱开始，天干和地支各往后推一位，依次排列。

例：月柱为丙寅，顺排大运为：丁卯 → 戊辰 → 己巳 → 庚午 → 辛未 → ...

天干顺推：甲→乙→丙→丁→戊→己→庚→辛→壬→癸→甲（循环）
地支顺推：子→丑→寅→卯→辰→巳→午→未→申→酉→戌→亥→子（循环）

### 逆排
从月柱开始，天干和地支各往前推一位，依次排列。

例：月柱为丙寅，逆排大运为：乙丑 → 甲子 → 癸亥 → 壬戌 → 辛酉 → ...

天干逆推：癸→壬→辛→庚→己→戊→丁→丙→乙→甲→癸（循环）
地支逆推：亥→戌→酉→申→未→午→巳→辰→卯→寅→丑→子→亥（循环）

## 四、起运年龄计算

### 顺排起运年龄
从出生日起，顺数到下一个**节**（注意是"节"而非"气"），计算总天数，除以 3，得数即为起运年龄。

公式：起运年龄 = 出生日到下一个节气天数 ÷ 3

### 逆排起运年龄
从出生日起，逆数到上一个**节**，计算总天数，除以 3，得数即为起运年龄。

公式：起运年龄 = 出生日到上一个节气天数 ÷ 3

### 十二节（用于起运计算）
立春、惊蛰、清明、立夏、芒种、小暑、立秋、白露、寒露、立冬、大雪、小寒

注：节与气的区别——每月一节一气，节为月之始，气为月之中。排大运只看"节"。

### 余数处理
- 余 1 天：约 4 个月起运
- 余 2 天：约 8 个月起运
- 余 0 天：整岁起运

四舍五入到最接近的整数岁数。

## 五、大运周期

每步大运管十年。

排列示例（假设 5 岁起运，月柱丙寅，顺排）：
| 大运序 | 年龄范围 | 干支 |
|--------|---------|------|
| 第一步 | 5-14 岁 | 丁卯 |
| 第二步 | 15-24 岁 | 戊辰 |
| 第三步 | 25-34 岁 | 己巳 |
| 第四步 | 35-44 岁 | 庚午 |
| 第五步 | 45-54 岁 | 辛未 |
| 第六步 | 55-64 岁 | 壬申 |
| 第七步 | 65-74 岁 | 癸酉 |
| 第八步 | 75-84 岁 | 甲戌 |

## 六、交运前的小运

小孩未交大运之前（即起运年龄之前），以月柱干支作为大运。

例：如果 5 岁起运，则 0-4 岁这段时间，以月柱的干支来分析运势。

## 七、流年

流年即每一年的天干地支，用来判断当年的运势吉凶。

- 流年干支与原局（四柱）产生的关系（冲、合、刑、害等）影响当年运势
- 流年与大运的关系也很重要：大运为环境，流年为引动
- 大运好、流年差：总体平稳但当年有波折
- 大运差、流年好：总体不利但当年有机遇

## 八、注意事项

1. **大运换运年**（交运前后1-2年）是人生重大转折的高发期，必须单独标注。乔布斯被逐(换运次年)、马云创业(换运次年)均在此窗口。
2. 大运天干管前五年，地支管后五年（此为一家之言，也有主张天干地支共管十年）
3. 大运与原局形成的天合地合、天克地冲等关系需要重点分析
4. 分析大运时要结合原局格局的喜忌，喜神到位则吉，忌神到位则凶
5. **岁运并临**（大运干支 = 流年干支）：极大概率触发人生重大事件，必须标注为 🔴 核心级信号。如毛泽东1976年丙辰大运遇丙辰流年。

## 九、经典格局的大运验证规律

经大量名人案例验证，以下格局-大运组合高度可靠：

| 格局 | 日主 | 特征 | 起飞大运 | 验证案例 |
|------|------|------|---------|---------|
| 伤官佩印 | 丁火戌月身弱 | 双印透干制伤官 | 中年行木火运（印比） | 马云 |
| 伤官生财 | 丁火戌月身弱 | 伤官透+偏财有根 | 中年行火运（比劫助身） | 马化腾 |
| 食神制杀 | 丙火 | 食神双透+七杀有制 | 木火运 | 乔布斯 |
| 从财格 | 壬水 | 无印比破格+财星三合 | 食伤木运（生火财） | 比尔盖茨 |
| 杀印相生 | 丁火子月 | 七杀当令+印星化杀 | 木火运（印比） | 毛泽东 |

**共性规律**：身弱喜木火者，中年入木火大运 = 人生爆发期。若原局格局清纯，爆发力加倍。分析时若命中此规律，标为 🔴 核心结论。

## 十、凶险格局警示（案例验证）

经牢狱、破产、早逝等负面案例验证，以下格局为高危信号，分析时必须标注 🔴：

| 危险格局 | 核心机制 | 可能结果 | 验证案例 |
|---------|---------|---------|---------|
| **枭神夺食** | 偏印克制食神 | 胡作非为、官非牢狱、克父 | 败家子两次死刑 |
| **比劫夺财无制** | 比肩劫财旺而无官杀制 | 暴富后破产、倾家荡产 | 一夜暴富后家破人亡 |
| **火炎土燥/四柱偏枯** | 无水润局，五行极端偏枯 | 福寿短、刑妻克子、牢狱 | 三次入狱、贩毒 |
| **伤官无制/伤官见官** | 伤官旺而**无印星克制**（区别于吉格「伤官佩印」）、或伤官直接克正官 | 目无法纪、铤而走险 | 走私判刑三年 |
| **日时相冲（冲夭煞）** | 日支与时支六冲 | 根基动摇，少年多灾甚至有短命风险 | 婴儿夭折案例 |
| **日支用神被双重克** | 日支为用神但被月支+时支克制 | 早逝/残疾 | 游泳溺亡、车祸 |
| **四柱同字旺极** | 地支全同一字且为忌神 | 「旺极犯岁」→ 夭折 | 四申金旺极三岁死 |

**分析时的使用规则**：
- 命中以上格局 → 标注 🔴，健康/安全建议必须明确且严肃
- 若危险格局有「解」→ 降为 🟡 并说明解法
- 不命中以上格局 → 不夸大健康风险
- **用神被灭法则**（经5例早逝案例验证）：当命局的用神**在天干透出被合/克 + 在地支的根被冲/刑/合走** 同时发生时，为最高级别危险信号 🔴🔴。这是李小龙、张国荣、科比、戴安娜、乔布斯五人在逝世年份的共同模式。
