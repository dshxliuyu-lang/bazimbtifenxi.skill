- 亥卯未合木局
- 寅午戌合火局

三合局需三字全见方成局，缺一则为半合（如申子半合水、子辰半合水），力量减半。

### 三会局（方局）

### 天地合（天地鸳鸯合）

天干地支同时相合，称为天地合。如：
- 甲子 + 己丑 = 天甲己合土 + 地子丑合土 → 天地合土
- 乙亥 + 庚寅 = 天乙庚合金 + 地寅亥合木
- 丁卯 + 壬戌 = 天丁壬合木 + 地卯戌合火

天地合是命局中极强的合化信号，常代表重大的人生契合（婚姻、合作等）。

## 空亡（旬空）

空亡又称"六甲空亡"，指天干地支循环中每旬缺少的两个地支。在空亡位置的地支力量减弱（虚浮无力），但也可能在某类命格中反而有利（物极必反）。

### 六甲旬空亡表

| 旬 | 天干 | 空亡地支 |
|------|------|---------|
| 甲子旬 | 甲子 → 癸酉 | 戌、亥 |
| 甲戌旬 | 甲戌 → 癸未 | 申、酉 |
| 甲申旬 | 甲申 → 癸巳 | 午、未 |
| 甲午旬 | 甲午 → 癸卯 | 辰、巳 |
| 甲辰旬 | 甲辰 → 癸丑 | 寅、卯 |
| 甲寅旬 | 甲寅 → 癸亥 | 子、丑 |

### 空亡速查（按日柱地支）

以日柱地支所属的旬查空亡：
- 日支：子、丑 → 空亡：寅、卯（日柱在甲寅旬）
- 日支：寅、卯 → 空亡：辰、巳（日柱在甲辰旬）
- 日支：辰、巳 → 空亡：午、未（日柱在甲午旬）
- 日支：午、未 → 空亡：申、酉（日柱在甲申旬）
- 日支：申、酉 → 空亡：戌、亥（日柱在甲戌旬）
- 日支：戌、亥 → 空亡：子、丑（日柱在甲子旬）

### 年空亡与日空亡
- **年空亡**：以年柱地支查，影响一生大环境
- **日空亡**：以日柱地支查，直接影响日主自身
- 年空亡和日空亡可同时存在，并不同步

### 空亡的解读
- **支空亡**：该支所在柱的力量打折（约减半）
- **吉神逢空**：好事落空、贵人难助力
- **凶神逢空**：坏事反而减轻
- **空亡逢冲**：冲可解空（但只是暂时填实）
- **空亡逢合**：合可填实

## 命宫推算

命宫是八字以外的辅助宫位，反映先天禀赋和人生大方向。

### 推算法（寅上起月法）

**步骤**：
1. 将月支对应数字：正月（寅）=1，二月（卯）=2，...，十二月（丑）=12
2. 将时支对应数字：子=1，丑=2，...，亥=12
3. **命宫地支** = 14 或 26 减去（月支数 + 时支数）
   - 若（月支数 + 时支数）< 14，则：命宫地支数 = 14 -（月支数 + 时支数）
   - 若（月支数 + 时支数）≥ 14，则：命宫地支数 = 26 -（月支数 + 时支数）
4. 命宫天干 = 以年干用"五虎遁元"法推该地支的天干

### 速查示例

月柱亥月（=10），时支卯（=4）：
- 10 + 4 = 14 ≥ 14 → 命宫地支数 = 26 - 14 = 12 → 亥
- 命宫在亥

### 参考
- 命宫在不同地支代表不同的先天倾向
- 命宫与大运流年发生冲合刑害时，亦有吉凶效应

## 胎元推算

胎元代表人出生前在母胎中的 10 个月，反映先天体质和根基。

### 推算方法

- **胎元天干** = 月干往后推 1 位（即月干的"母亲"——生月干之干）
- **胎元地支** = 月支往后推 10 位（或月支往前推 2 位），即从月支始，顺数到第 10 个月

### 示例

月柱丁亥：
- 胎元天干 = 丁的前一位 → 丙（实际上是甲生丁，所以胎元天干为甲？）

更直接的方法：胎元 = 月柱干支各往前推一位
月柱丁亥 → 胎元戊子（天干往前一位：丁→戊；地支顺推：亥→子）

再例：月柱丙寅 → 胎元丁卯

## 十神力量统计方法

用于计算命局中十神的分配比例（百分比），参照 bazi-calculator.com 的方法。

### 计算元素
纳入统计的来源：
1. 四柱天干（4 个）
2. 四柱地支藏干（按权重：本气 ×1.0 + 中气 ×0.5 + 余气 ×0.25）

### 计算步骤
1. 列出所有纳入的天干和藏干
2. 确定每个干相对于日主是什么十神
3. 同级十神相加（如所有正官、所有七杀...）
4. 计算百分比 = 该十神权重 ÷ 总权重 × 100%
5. 按五大类汇总：比劫、食伤、财星、官杀、印星

### 输出格式
```
十神分布：
- 印星（金）：11%（正印11%、偏印0%）
- 比劫（水）：58%（比肩15%、劫财43%）
- 食伤（木）：25%（食神3%、伤官21%）
- 财星（火）：7%（正财7%、偏财0%）
- 官杀（土）：0%（正官0%、七杀0%）
```

## 日主旺衰定量计算

基于 bazi-calculator.com 的现代藏干方法，计算 **生扶者** vs **损抑者** 的百分比。

### 计算规则
1. **生扶日主的力量** = 正印 + 偏印 + 比肩 + 劫财（所有生我和同我的力量）
2. **损抑日主的力量** = 食神 + 伤官 + 正财 + 偏财 + 正官 + 七杀（所有我生、我克、克我的力量）
3. 各项按藏干权重计（本气 ×1.0, 中气 ×0.5, 余气 ×0.25）
4. 月令加权：月支藏干力量加倍（×1.5）

### 旺衰判断标准
- **生扶 > 60%** → 身旺
- **生扶 40%-60%** → 中和
- **生扶 < 40%** → 身弱
- **生扶 > 80%** → 考虑从强格
- **生扶 < 20%** → 考虑从弱格

### 注意事项
- 定量计算为辅助参考，不能完全替代传统旺衰判断
- 合化后的五行变化会影响实际力量分布
- 最终判断需结合得令、得地、得势的综合分析

### 从格判定补充（案例验证）

判断从弱格需同时满足：
1. 生扶力量 < 20%
2. **原局无印星或比劫透干破格**（印比为忌但若透干有根，则假从或不从）
3. 月令为财/官/食伤且当令

常见从格速查：
- **从财格**：日主无根+财星三合局+无印比透干。喜食伤生财，忌印比。例：比尔盖茨(壬水日主+双丙火+午戌火局+无印)
- **从杀格**：日主无根+官杀成局+无印化杀。喜财生杀，忌印。例：周杰伦(若从格成立)
- **从儿格**：日主无根+食伤成局+无印制食伤。喜比劫生食伤，忌印。

假从格（生扶 15-25% 且印比有微根）：大运顺则为真从，大运逆则按身弱论。

## 真太阳时校正

真太阳时（Real Solar Time, RST）是排盘时的重要校正，尤其对于出生在非标准时区中心的用户。

### 计算要素
1. **经度校正**：每偏离时区中心 1° 经度，时间差约 4 分钟
   - 东经 120° 以东的地区，真太阳时比北京时间早
   - 东经 120° 以西的地区，真太阳时比北京时间晚
2. **均时差**：地球公转速度不均造成的微小偏差（全年在 ±16 分钟之间，一般可忽略）

### 中国主要城市经度参考
| 城市 | 经度 | 校正值（相对北京时间 120°E） |
|------|------|--------------------------|
| 北京 | 116.4°E | -14.4 分钟 |
| 上海 | 121.5°E | +6 分钟 |
| 广州 | 113.3°E | -26.8 分钟 |
| 成都 | 104.1°E | -63.6 分钟（约 -1 小时） |
| 哈尔滨 | 126.6°E | +26.4 分钟 |
| 乌鲁木齐 | 87.6°E | -129.6 分钟（约 -2 小时） |

### 校正公式
真太阳时 = 北京时间 + (当地经度 - 120°) × 4 分钟

### 分析中的使用
- 若出生时间接近时辰分界点（如 04:55 或 07:05），真太阳时校正可能导致时辰变化，应提示用户
- 中国横跨多个时区，西部地区（新疆、西藏等）校正尤为重要
- 分析时应标注"北京时间 X 时，经校正后真太阳时为 Y 时"
