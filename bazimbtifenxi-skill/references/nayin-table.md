# 六十甲子纳音表

## 纳音概述

纳音是把六十甲子与五行（木火土金水）及五音（宫商角徵羽）相结合的一套体系。每个天干地支组合配一个纳音五行，共三十组，每组两个甲子。

纳音用于补充分析命局的深层气质，尤其在婚配合婚、日柱特性分析中常用。

## 纳音在分析中的使用

1. **日柱纳音**：反映本命气质，可用于婚配合婚（纳音相生为吉，相克需注意）
2. **年柱纳音**：反映家族、祖上之气质底色
3. **各柱纳音**：补充五行判断，如命局缺火但年柱纳音为火，则火气暗藏
4. **纳音相生**：相邻两柱纳音相生（如年柱木生月柱火），气场流通顺畅
5. **纳音相克**：相邻两柱纳音相克，需看有无通关
