## 神煞使用原则

### 查法优先级
1. **年支查** — 最常用，影响一生大局
2. **日干查** — 十神类神煞以此为主
3. **月支查** — 天德、月德、龙德等月令神煞
4. **日支查** — 夫妻宫相关神煞

### 使用注意事项
1. **不可喧宾夺主**：神煞为辅助参考，必须先论格局旺衰和用神，再参看神煞
2. **逢冲则减力**：吉神被冲克，其力大减；凶神被冲克，其力亦减
3. **多处出现则加强**：同一神煞在命局多处出现，其应验度更高
4. **配合十神解读**：桃花 + 伤官 = 风流才子型；华盖 + 正印 = 学者型
5. **标注 `*` 的星**：仅部分流派使用，非通用

### Bazi-Calculator 神煞标注
在分析输出中，部分神煞会用 `*` 标记（如 流霞*、血刃*、孤辰*），表示这些神煞并非所有命理流派都采用。分析时应说明争议性。

### 三刑的特殊性（案例修正）

三刑（丑未戌三刑、寅巳申三刑、子卯刑）**非纯粹凶信**。经乔布斯(丑未戌三刑年: 1976创立苹果 + 1997重返苹果)、马云等案例验证：

- 三刑年 = **高压 + 转机并存**，是「破局之年」
- 吉神到位 → 从压力中突破、绝地反弹
- 忌神到位 → 被压垮、一蹶不振
- 分析时标注 🔴「刑动之年，吉凶参半，有破局之机也有受压之险」
