## 简介

Bazi Skill 是运行在 Claude Code 上的八字命理分析引擎。通过交互式对话收集出生信息，排出四柱八字，参照《穷通宝典》《三命通会》《滴天髓》《渊海子平》《子平真诠》等九本经典典籍进行专业分析。

**八字不是判词，是说明书。每个「缺点」都是放错地方的优点。**

---

## 功能

- 🏯 **八字排盘** — 四柱、十神、藏干、纳音、十二长生、空亡
- 🔮 **大运流年** — 顺逆排运、起运年龄、流年分析
- 🧠 **MBTI 人格映射** — 潘子端 1937 体系 + 五行法，八字↔16型人格交叉验证
- 💑 **合婚配对** — 七大维度综合评分
- 🔧 **五行补救** — 颜色/方位/行业/饰品/饮食/习惯
- 🐉 **生肖分析** — 生肖特性 + 犯太岁
- 📅 **流月/每日运势**

## 安装

```bash
# 安装到 Claude Code（全局可用）
mkdir -p ~/.claude/skills
git clone https://github.com/<dshxliuyu-lang>/bazimbtifenxi-skill.git ~/.claude/skills/bazi
```

安装后在 Claude Code 中输入「八字」即可触发。
